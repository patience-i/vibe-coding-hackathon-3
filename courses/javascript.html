JavaScript Tutorial

Learn the programming language of the web

What is JavaScript?

  • JavaScript is the programming language of the Web
  • JavaScript can update and change both HTML and CSS
  • JavaScript can calculate, manipulate and validate data
  • JavaScript enables interactive web pages

A Simple JavaScript Example

Example
<!DOCTYPE html> <html> <head> <title>JavaScript Example</title> </head> <body> <h1>My First JavaScript</h1> <button onclick="document.getElementById('demo').innerHTML = 'Hello JavaScript!'">Click me!</button> <p id="demo">This text will change.</p> </body> </html>

JavaScript Can Change HTML Content

One of many JavaScript HTML methods is getElementById().

This example uses the method to "find" an HTML element (with id="demo") and changes the element content (innerHTML) to "Hello JavaScript":

document.getElementById("demo").innerHTML = "Hello JavaScript";

JavaScript accepts both double and single quotes:

document.getElementById('demo').innerHTML = 'Hello JavaScript';

JavaScript in <head> or <body>

You can place any number of scripts in an HTML document.

Scripts can be placed in the <body>, or in the <head> section of an HTML page, or in both.

It's a good practice to place scripts at the bottom of the <body> element to improve page load performance.
