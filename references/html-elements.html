HTML Elements Reference

A comprehensive guide to HTML elements, attributes, events, and colors.

HTML Elements

Tag Description Example
<a> Defines a hyperlink <a href="https://example.com">Visit Example</a>
<div> Defines a division or section in an HTML document <div>This is a division.</div>
<h1> Defines a level 1 heading <h1>This is a heading</h1>
<p> Defines a paragraph <p>This is a paragraph.</p>
<img> Defines an image <img src="image.jpg" alt="Description">

HTML Attributes

Attribute Description Example
href Specifies the URL of the page the link goes to <a href="https://example.com">Example</a>
class Specifies one or more class names for an element <div class="container">Content</div>
id Specifies a unique id for an element <div id="header">Header</div>

HTML Events

Event Description Example
onclick Executed when the element is clicked <button onclick="myFunction()">Click me</button>
onload Executed when the page has finished loading <body onload="myFunction()">
onchange Executed when the value of an element has changed <input onchange="myFunction()">
