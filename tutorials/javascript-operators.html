JavaScript Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

We will look at the following types of operators:

  • Arithmetic operators
  • Assignment operators
  • Comparison operators
  • Logical operators
  • Type operators

JavaScript Arithmetic Operators

Arithmetic operators perform arithmetic on numbers (literals or variables).

Operator Description Example
+ Addition 5 + 2 = 7
- Subtraction 5 - 2 = 3
* Multiplication 5 * 2 = 10
/ Division 5 / 2 = 2.5
% Modulus (Remainder) 5 % 2 = 1
++ Increment let x = 5; x++; // x is now 6
-- Decrement let x = 5; x--; // x is now 4
** Exponentiation (ES2016) 5 ** 2 = 25
Example:
// Arithmetic operators let a = 10; let b = 3; let addition = a + b; // 13 let subtraction = a - b; // 7 let multiplication = a * b; // 30 let division = a / b; // 3.33... let modulus = a % b; // 1 let exponentiation = a ** b; // 1000 (10^3) // Increment and decrement let c = 5; c++; // c is now 6 let d = 5; d--; // d is now 4

JavaScript Assignment Operators

Assignment operators assign values to JavaScript variables.

Operator Example Same As
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
**= x **= y x = x ** y

// Assignment operators let x = 10; x += 5; // x = x + 5; (x is now 15) x -= 3; // x = x - 3; (x is now 12) x *= 2; // x = x * 2; (x is now 24) x /= 4; // x = x / 4; (x is now 6) x %= 4; // x = x % 4; (x is now 2) x **= 3; // x = x ** 3; (x is now 8)

JavaScript Comparison Operators

Comparison operators compare two values and return a boolean value (true or false).

Operator Description Example
== Equal to (value only) x == y
=== Equal to (value and type) x === y
!= Not equal (value only) x != y
!== Not equal (value and type) x !== y
> Greater than x > y
< Less than x < y
>= Greater than or equal to x >= y
<= Less than or equal to x <= y
? Ternary operator condition ? exprIfTrue : exprIfFalse
Example:
// Comparison operators let a = 5; let b = "5"; let c = 8; console.log(a == b); // true (value is equal) console.log(a === b); // false (value is equal but type is different) console.log(a != c); // true (value is not equal) console.log(a !== b); // true (type is different) console.log(c > a); // true console.log(c < a); // false console.log(a >= 5); // true console.log(c <= 5); // false // Ternary operator let age = 20; let canVote = age >= 18 ? "Yes" : "No"; console.log(canVote); // "Yes"

JavaScript Logical Operators

Logical operators are used to determine the logic between variables or values.

Operator Description Example
&& Logical AND (x && y) is true if both x and y are true
|| Logical OR (x || y) is true if either x or y is true
! Logical NOT !x is true if x is false
Example:
// Logical operators let x = 6; let y = 3; // AND operator console.log(x > 5 && y < 5); // true (both conditions are true) console.log(x > 10 && y < 5); // false (first condition is false) // OR operator console.log(x > 5 || y > 5); // true (first condition is true) console.log(x < 5 || y > 5); // false (both conditions are false) // NOT operator console.log(!(x > 5)); // false (because x > 5 is true) console.log(!(y > 5)); // true (because y > 5 is false)

JavaScript Type Operators

Type operators are used to check the type of a variable or expression.

Operator Description Example
typeof Returns the type of a variable typeof x
instanceof Returns true if an object is an instance of an object type x instanceof Array
Example:
// Type operators let str = "Hello"; let num = 123; let bool = true; let arr = [1, 2, 3]; let obj = {name: "John", age: 30}; let func = function() { return "Hello"; }; let nullVar = null; let undefinedVar; console.log(typeof str); // "string" console.log(typeof num); // "number" console.log(typeof bool); // "boolean" console.log(typeof arr); // "object" console.log(typeof obj); // "object" console.log(typeof func); // "function" console.log(typeof nullVar); // "object" (this is a known JavaScript quirk) console.log(typeof undefinedVar); // "undefined" // instanceof operator console.log(arr instanceof Array); // true console.log(obj instanceof Array); // false
