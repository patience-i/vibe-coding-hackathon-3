What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements that tell the browser how to display the content.

HTML elements are represented by tags, which are enclosed in angle brackets. Most elements have an opening tag and a closing tag.

Basic HTML Document Structure

Every HTML document follows a basic structure that includes several mandatory elements:

<!-- This is a comment in HTML -->
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

Let's break down this structure:

  • <!DOCTYPE html> - Declares the document type and HTML version
  • <html> - The root element of an HTML page
  • <head> - Contains meta information about the page (not visible on the page)
  • <title> - Specifies a title for the page (shown in the browser tab)
  • <body> - Contains the visible page content
  • <h1> - Defines a large heading
  • <p> - Defines a paragraph

HTML Headings

HTML provides six different heading levels, from <h1> (most important) to <h6> (least important).

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

This is Heading 6

Headings are important for both structure and SEO (Search Engine Optimization). Use them logically to create a hierarchy of content on your page.

HTML Paragraphs

Paragraphs are defined with the <p> tag and represent blocks of text.

<p>This is a paragraph. It represents a block of text that is separated from other blocks of text by a blank line.</p>
<p>This is another paragraph. HTML automatically adds some space before and after paragraphs.</p>

This is a paragraph. It represents a block of text that is separated from other blocks of text by a blank line.

This is another paragraph. HTML automatically adds some space before and after paragraphs.

HTML Links

Links allow users to click and navigate to other pages. They are created with the <a> tag.

<a href="https://www.example.com">Visit Example.com</a>
<a href="contact.html">Contact Us</a>
Visit Example.com
Contact Us

The href attribute specifies the destination of the link. It can be a URL (absolute link) or a path to another file in your website (relative link).

HTML Images

Images make your website visually appealing. Add them using the <img> tag.

<img src="image.jpg" alt="Description of the image" width="300" height="200">

The src attribute specifies the path to the image, and the alt attribute provides alternative text for accessibility and SEO. The width and height attributes set the dimensions of the image.

Note: The <img> tag is a self-closing tag, meaning it doesn't need a closing tag.

HTML Lists

HTML provides two main types of lists: ordered lists and unordered lists.

Unordered Lists

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

Ordered Lists

<ol>
    <li>First Step</li>
    <li>Second Step</li>
    <li>Third Step</li>
</ol>
  1. First Step
  2. Second Step
  3. Third Step
